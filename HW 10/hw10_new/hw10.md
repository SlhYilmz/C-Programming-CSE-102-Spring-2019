## Overview

This assignment consists of four sub-parts. For each part, you must perform performance comparisons in terms of execution time (in milliseconds) for three data structures: **Stack**, **Queue**, and **Binary Search Tree (BST)**.

### Constraints

- **Immutable Prototypes:** Do not change function names, parameters, or structure names. Any unauthorized changes will result in zero credit for that part.

## Part 1. [20pts] Structure Filling

Write a function that takes three data structure as parameters and a data array to fill them with the values of the data array. The data structures should be queue, stack and binary search tree. Tract the execution time of filling time for each data structure and report the total execution time of filling the structures separately.

**Function Prototype:**
`void fill_structures(stack * stack, queue * queue_, bst * bst, int data[20])`

---

## Part 2. [30pts] Ordered Print

Write a function that takes the data structures which were filled on part one as parameters and print them in descending order. Report the total of ordering and printing time for each structure separately.

**Function Prototype:**
`void ordered_print(stack * stack, queue * queue_, bst * bst_)`

---

## Part 3. [20pts] Search

Write a function to search for a specific integer within the structures. The function must print all found items and the number of steps taken to find them. If no result is found, it should report that.

**Example:**
If a queue is `(tail)[1, 3, 5, 2, 11, 18](head)` and you search for `3`, the output should be: `"1 result founded on 5. step."`

**Function Prototype:**
`void search(stack * stack, queue * queue_, bst * bst, int val_to_search)`

---

## Part 4. [30pts] Special Traversal

Write a function that prints the structures using a specific traversal method: Start with the **maximum** value, then the **minimum**, then the next maximum, then the next minimum, and so on.

**Example:**
If a stack is `3, 5, 6, 2, 1, 7 (top)`, the output should be: `7, 1, 6, 2, 5, 3`.

Report the total of traverse time for each structure separately. 

**Function Prototype:**
`void special_traverse(stack * stack, queue * queue_, bst * bst_)`
